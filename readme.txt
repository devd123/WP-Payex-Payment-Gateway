==== WP Payex Payment Gateway ====
Contributors: neeru & dev company
tags: payment gateway, payex , simple payment
Requires at least: 3.5
Tested up to: 4.4.2

A payment gateway for the payex gateway support for the all wordpress site user

== Description ==
setup payex payment gateway for the wordpress website and make payment with the order listing table in wp backend.
plugin will have a form with the fields like name , billing adress , city , phone , email id etc.

[Add_Payment_Page] - the short will be used for the payment form page 

Admin can save their payex account setting at the plugin setting page in the wp backend
Also user listing will be create in the wp backend.
